R(R08) Prueda del seis ¶

  1. ¿Que sitio es éste? (Identificador del sitio)

    Se trata del sitio web Skeleton's Trap, este nombre es dado por el equipo de clash royale al que va dirigido. En todo momento aparece el nombre Skeleton's Trap arriba a la izquierda

  2. ¿En que página estoy? (Nombre de la página)

    En la parte del frontend hay migas de pan que te muestran donde estás ubicado, aunque realmente no hay mucha jerarquia de páginas.

    Por la parte del backend puedes ver donde estas porque se marca con un resaltado más oscuro en el menú en la página donde estes.

  3. ¿Cuales son las principales secciones del sitio? (Secciones)

    Las secciones de la web se encuentran arriba en el menu. Cuando el usuario se loguea puede ver todas las opciones del menú.

    En la parte del backend solo puede ver aquellas a las que tiene permiso.

  4. ¿Que opciones tengo en este nivel? (Navegación local)

    La navegación local dentro de cada nivel del sitio siempre se realiza a través de botones o enlaces.

  5. ¿Donde estoy en el esquema de las cosas? (Indicadores de "Usted está aquí")

    La web dispone en todo momento de "migas de pan" para tener al usuario informado de donde se encuentra dentro del mapa de la web.

  6. ¿Como busco algo? (Buscador)

    La web no necesita un buscador ya que no tiene contenido necesario para ello. A si que el usuario se puede guiar a través de las diferentes partes del menú ya que es una aplicación muy visual.
